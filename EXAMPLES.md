# Example Commands

Here are some example commands you can try with the browser agent:

## Navigation

```
go to github.com
navigate to https://news.ycombinator.com
open reddit.com
```

## Search and Interaction

```
search for "machine learning" on Google
click on the first search result
scroll down the page
scroll up
go back
refresh the page
```

## Form Filling

```
find the search box and type "AI news"
fill in the email field with user@example.com
enter "John Doe" in the name field
click the submit button
```

## Information Extraction

```
what's the title of this page?
read the main heading
get the text from the first paragraph
list all the links on this page
```

## Screenshots and Visual

```
take a screenshot
capture a screenshot of the page
take a picture of the current page
```

## Advanced Actions

```
hover over the menu button
right click on the logo
press Enter key
wait 2 seconds
execute: document.title (run JavaScript)
```

## Multi-Step Tasks

```
Go to Amazon, search for "laptop", and click on the first result

Navigate to Wikipedia, search for "Artificial Intelligence", 
and take a screenshot of the article

Open YouTube, search for "coding tutorials", 
and click on the first video
```

## Browser State

```
what page am I on?
show me the current URL
what's loaded in the browser?
```

## Tips for Better Results

1. **Be Specific**: Instead of "click the button", say "click the submit button" or "click the blue login button"

2. **Use Selectors When Needed**: If the AI struggles, you can be more specific like "click the button with text 'Sign In'"

3. **Break Down Complex Tasks**: For multi-step operations, you can either:
   - Describe the full task in one command
   - Break it into multiple commands

4. **Wait When Needed**: If a page is loading slowly, say "wait 3 seconds" before the next action

5. **Check State**: Ask "what page am I on?" to verify where you are

6. **Use Natural Language**: The AI understands context, so speak naturally!
